# Daily WhatsApp Issue Summary Update SOP (Machine Issue Status)

---

## Objective

To ensure customers receive a clear daily update on machine issues, actions taken, and current status.

---

## Responsibility

The **Customer Support Executive** must send a **daily summary update in the customer WhatsApp group before end of day**.

---

## Daily Update Template (Standard Format)

Use this format for every update:

Machine No: Issue
Action Taken:
Target Date (if pending):

---

## Example (Pending Issue)

Machine 29: No signal from camera
Action Taken: Checked GPU port and restarted system
Target Date: 15 Feb 2026

---

## If Issue is Completed

Use this format:
Machine 29: No signal from camera
Action Taken: Camera cable replaced
Completed Date: 14 Feb 2026


---

## Guidelines

- Daily summary must be shared even if there is **no major update**
- Use **simple and clear language**
- Always mention the **machine number**
- Do not skip the **current status**
- Do not close the issue without customer confirmation
