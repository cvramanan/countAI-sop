# Testing Requirements SOP

---

## 1. Approval Sheet Requirement

Testing must start **only after receiving the approval sheet**.

### Approval Sheet must include:

- Cabinet version and Quantity  
- Software version  
- Machine model and Type  
- Client name  
- Special instructions  

⚠️ **Without approval, testing must NOT begin.**

---

## 2. Material Shortage During Testing

### Condition:
If the Production Team does not provide sufficient or required materials during testing.

### Procedure:

- Testing Engineer shall check material availability before starting testing.  
- If material shortage occurs during testing, testing must be **stopped immediately**.  
- Testing Engineer shall inform the Production Team about the material shortage.  
- Record the testing stop reason in the testing report.  
- Testing shall resume only after required materials are provided by the Production Team.  

### Engineering Change Control

Any hardware or software modification must be processed through:

- **Engineering Change Request (ECR)**
- **Engineering Change Notice (ECN)**

After ECN implementation:

- Mandatory retesting shall be conducted to ensure product conformity.

---

## 3. Software Installation and Version Control

- Install approved current software version before testing.  

### If Software Version is Upgraded After Testing Completion:

- Cabinet must be re-tested.  
- Minimum re-testing time: **5 hours per cabinet**  
- Packing and dispatch must be stopped until retesting is completed.  

### Verify:

- Software installation success  
- System boot and stability  
- Communication functionality  
- Software features working properly  

### Software Upload Time:

- 6 hours per cabinet  

### Additional Verification:

- All system functions  
- Hardware and software integration  
- Display and communication  

---

## 4. Testing Time Requirement

| Quantity | Testing Time |
|-----------|-------------|
| 1 Cabinet | 8 hours (Software Test + Hardware Testing) |
| 15 Cabinets | 2 days total testing |
| New Issue Found | Additional 3 hours per cabinet |
| Software Update Retesting | Minimum 6 hours per cabinet |

---

## 5. Issue Handling Procedure

If any issue is found:

- Stop testing immediately  
- Record issue details  
- Inform Production & Software Team  
- Resolve the issue  
- Perform full retesting  

---

## 6. Acceptance Criteria

Cabinet is accepted only if:

- Electrical test passed  
- Software test passed  
- Functional test passed  
- No pending software update  
- QC approval completed
