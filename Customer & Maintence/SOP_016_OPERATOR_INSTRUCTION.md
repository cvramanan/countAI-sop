# OPERATOR’s SOP

---

# 1. Routine Camera Cleaning

Follow the scenarios below:

---

## Scenario 1.1  
### Dust Observed on Live Screen

- If dust is visible on the live monitoring screen:
  - Stop and inspect the camera immediately.
  - Clean the camera lens properly using approved cleaning materials.
  - Ensure the image is clear before restarting operation.

---

## Scenario 1.2  
### Camera Dust Captured as Hole

- If camera dust is detected as a hole defect:
  - Pause inspection.
  - Clean the camera lens carefully.
  - Verify whether the defect still appears.
  - Confirm defect authenticity before resuming production.

---

## Scenario 1.3  
### Roll Completion

- Clean the camera when a roll is completed.
- Ensure the lens is free from:
  - Dust
  - Oil
  - Lint particles
- Confirm clear visibility before starting the next roll.

---

# 2. Roll Reset Procedure

- **Roll Reset must be pressed manually every time a roll ends.**
- If the fabric roll is cut before reaching full counter:
  - Press **“Roll Reset”** manually.
  - Example: Roll cut due to excessive defects.

⚠️ Do not forget manual reset in partial roll situations.

---

# 3. Defect Correction During Alarm

- During an alarm:
  - Press **Thumbs Up / Thumbs Down** correctly.
  - Ensure proper defect confirmation.
  - Refer to **Page 3** for guidance.

Incorrect feedback affects system accuracy.

---

# 4. Slit Line / Honeycomb Program Handling

Whenever **“Slit Line”** or **“H.Comb”** program is running:

- Create the defect on the **Sensor Trigger Magnet Side** (Green Arrow).
- Ensure defect is created in a **perfect straight line** (Red Arrow).
- Accuracy is mandatory.
- Refer to machine reference images below.

---

# 5. Machine-Specific Notes

## Mayer Machine

- Follow machine-specific handling guidelines.
- Ensure proper sensor alignment.

## Pilotelli Machine

- Follow machine-specific alignment procedures.
- Confirm straight-line defect creation.

---

# 6. Manual Roll Reset (Mid-Roll Cut)

If roll is cut before reaching exact counter:

- Reset roll manually.
- Ensure correct counter restart.
- Verify system reflects updated roll number.

---

# 7. Roll Counter Comparison

- Roll counter comparison data must be filled manually.
- Entry must be done at the beginning of each roll.
- Ensure accuracy between:
  - Machine counter
  - System counter

---

# Operator Responsibility

The operator is responsible for:

- Camera cleanliness  
- Correct roll reset  
- Proper alarm response  
- Accurate defect creation  
- Roll counter data accuracy  

Failure to follow this SOP may result in incorrect defect detection and reporting errors.
