# KNITI – Supernova UI Usability Training
## Standard Operating Procedure (SOP)

---

## 1. Purpose
To define the standard procedure for operating the **KNITI Supernova UI system** for:
- Machine monitoring  
- Defect detection  
- Roll management  
- Camera control  

---

## 2. Scope
This SOP applies to:
- Machine Operators  
- Supervisors  
- Quality Inspectors  
- Maintenance Team  

---

## 3. Responsibilities

| Role | Responsibility |
|------|---------------|
| Machine Operator | Enter program details, monitor defects, authenticate alarms |
| Supervisor | Verify defect logs, control defect detection settings |
| Maintenance | Manage camera settings, cleaning intervals, bypass control |

---

# 4. Procedure

---

# 4.1 Home Page Operation

## 4.1.1 Machine Monitoring

Verify the following on the Home Screen:

1. **Machine Status** – Running / Not Running  
2. **Doff Count** – Current roll & total doff count  
3. **Machine Name** – Confirm correct machine ID  
4. **Roll Number** – Verify active roll number  
5. **Fabric Type** – Confirm fabric loaded  
6. **Defect Count** – Check defect type and quantity  

---

## 4.1.2 Indication Buttons

Check status indicators:

- Bypass Status  
- Camera Status  
- Hardware Status  
- Software Status  

> ⚠ If any abnormal status is shown, inform maintenance immediately.

---

## 4.1.3 Start New Roll

1. Click **START NEW ROLL**
2. Confirm roll reset
3. Verify defect count reset
4. Ensure correct roll number is updated

---

# 4.2 Defect Alarm Authentication

When a defect is detected:

1. Alarm image popup appears.
2. Review defect image carefully.
3. Select appropriate option:

- ✅ **TRUE ALARM** – If defect is genuine  
- ❌ **FALSE ALARM** – If wrongly captured  

> ⚠ Every alarm must be authenticated immediately.

---

# 4.3 Program Information Entry

Navigate to: **PROGRAM**

Enter details from Knit Card:

| Field | Action |
|-------|--------|
| Knit Type | Enter fabric type |
| Fabric Details | Enter yarn details |
| Cotton Count | Enter from Knit Card |
| Polyester Count | Enter if applicable |
| Lycra | Select Yes / No |
| Fabric Program | Enter program number |
| Feeder Type | Select feeder type |
| GSM | Enter GSM |
| LOT | Enter Knit Card number |
| LL | Enter Loop Length |
| Total Revolution | Enter revolution per roll |
| Roll Weight | Enter roll weight |
| Reset Roll | Enter new roll number (if required) |
| JOB Number | Enter booking number |
| Party | Enter buyer details |
| Color | Enter fabric color |

After completion:
Click SUBMIT


> ✔ Verify all entries before submission.

---

# 4.4 Defect Log Management

Navigate to: **DEFECT LOG**

## 4.4.1 Viewing Defects

- Use arrow buttons to scroll images.
- Review:
  - Defect Name  
  - Roll Number  
  - Time of Defect  

---

## 4.4.2 Filtering Defect Data

1. Click **Filter Icon**
2. Select:
   - Date  
   - Roll Number  
   - Defect Type  
3. Click **Submit**

---

# 4.5 Defect Control

Navigate to: **DEFECT CONTROL**

To stop capturing specific defects:

1. Toggle ON/OFF required defect types.
2. Click **Submit**.

> ⚠ Only Supervisor authorization allowed for disabling defects.

---

# 4.6 Settings Management

Navigate to: **SETTINGS**

---

## 4.6.1 Change Language

1. Click **Change Language**
2. Select preferred language
3. Confirm selection

---

## 4.6.2 Shift Time Configuration

1. Click **Shift Time**
2. Edit:
   - Shift Name  
   - Start Time  
   - End Time  
3. Click **Update / Save**

---

## 4.6.3 Bypass Control

1. Click **Bypass**
2. Toggle ON/OFF as required

> ⚠ Bypass should only be used during maintenance or calibration.

---

## 4.6.4 Live Camera Operation

### Open Live Camera


### Change Camera
1. Click anywhere on live image
2. Select Camera (Cam 1 / Cam 2)

### Adjust Exposure & Gain
1. Click on live image
2. Enter Exposure value
3. Enter Gain value
4. Click Submit

### Close Live Camera


---

## 4.6.5 Camera Cleaning Interval

1. Go to **Camera Cleaning**
2. Select:
   - Day
   - Hour
3. Click **Submit**

> Recommended: Clean every 24–48 hours.

---

# 5. Safety & Control Points

- Always authenticate defect alarms immediately.
- Do not disable defect detection without authorization.
- Do not enable bypass during production without approval.
- Ensure program details are entered before production start.
- Maintain proper camera cleaning schedule.

---
